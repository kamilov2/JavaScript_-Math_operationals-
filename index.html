Matematik amallar va Math obyekti

Matematik amallar

Matematik amallar bo'yicha alohida keltirib tushuntirilgan qo'llanma.

        
// Matematik amallar
let x = 5;
let y = 1000;
let sum = x + y; // Qo'shish
let p = x * y; // Ko'paytirish
let s = x / y; // Bolish
let o = x % y; // Foyizga bolish
let a = x - y; // Ayirish

console.log("Qo'shish:", sum);
console.log("Ko'paytirish:", p);
console.log("Bolish:", s);
console.log("Foyizga bolish:", o);
console.log("Ayirish:", a);

Math obyekti

`Math` obyektini tushunish bo'yicha alohida keltirib tushuntirilgan qo'llanma.

        
// Math obyekti
let x = 10;
let y = 50;
console.log("Eng katta son:", Math.max(x, y)); // Eng katta son
console.log("Eng kichik son:", Math.min(x, y)); // Eng kichik son
console.log("Random son:", Math.random()); // Tasodifiy son
console.log("Floor son:", Math.floor(Math.random() * 50) + 1); // Eng kichik butun son
console.log("Ceil son:", Math.ceil(Math.random() * 50) + 1); // Eng katta butun son
console.log("Round son:", Math.round(Math.random() * 50) + 1); // Eng yaqin butun son
console.log("Absolut son:", Math.abs(-10)); // Absolut qiymat
